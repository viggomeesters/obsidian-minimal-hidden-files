    ".claude/commands/example.md",

    ".gitignore",
    ".mcp.json"
  ],

    ".codex"
  ],
  "deniedPresent": []

Denied paths `.obsidian`, `.trash`, and `.git` were not present in the revealed file/folder state.

Native File Explorer DOM contained these dot entries:

```text
.env
.gitignore
.mcp
.claude
.codex
```

Obsidian may strip extensions from file title labels in the DOM, so `.mcp` corresponds to `.mcp.json`.
